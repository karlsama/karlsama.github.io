# Java 实现学生成绩管理系统

> 一个完整的练手项目，涵盖：类设计、集合操作、控制台交互、异常处理、成绩校验

## 一、项目功能

| 序号 | 功能 | 说明 |
|------|------|------|
| 1 | 添加学生 | 输入学号、姓名、成绩（0-100自动校验） |
| 2 | 查看所有学生 | 显示所有学生信息 + 等级 |
| 3 | 查询学生 | 按学号查询单个学生 |
| 4 | 删除学生 | 按学号删除学生 |
| 5 | 统计平均分 | 计算所有学生的平均成绩 |
| 6 | 退出 | 退出系统 |

## 二、技术点总结

| 技术 | 在本项目中的应用 |
|------|------------------|
| 封装 | `Student` 类的私有属性 + getter/setter |
| 集合 | `ArrayList<Student>` 存储学生对象 |
| 循环 | `while(true)` 显示菜单直到退出 |
| 分支 | `if-else` 处理用户选择 |
| 异常处理 | `try-catch` 防止输入类型错误 |
| 校验逻辑 | 成绩必须在 0-100 之间 |

## 三、代码实现

### 3.1 Student.java（学生类）

```java
package org.example.practice;

public class Student {
    private int id;       // 学号
    private String name;  // 姓名
    private double score; // 成绩

    // 构造方法
    public Student(int id, String name, double score) {
        this.id = id;
        this.name = name;
        setter(score);  // 调用setter做校验
    }

    // getter
    public int getId() { return id; }
    public String getName() { return name; }
    public double getScore() { return score; }

    // setter（带校验）
    public void setter(double score) {
        if (score >= 0 && score <= 100) {
            this.score = score;
        } else {
            System.out.println("成绩必须在0-100之间，设为默认值0");
            this.score = 0;
        }
    }

    // 根据成绩返回等级
    public String getGrade() {
        if (score >= 90) return "A";
        else if (score >= 80) return "B";
        else if (score >= 70) return "C";
        else if (score >= 60) return "D";
        else return "F";
    }

    // 显示学生信息
    public void showInfo() {
        System.out.println("学号：" + id +
                "，姓名：" + name +
                "，成绩：" + score +
                "，等级：" + getGrade());
    }
}
### 3.2 StudentManager.java（主程序）

```java
package org.example.practice;

import java.util.ArrayList;
import java.util.Scanner;

public class StudentManager {
    public static void main(String[] args) {
        ArrayList<Student> students = new ArrayList<>();
        Scanner scanner = new Scanner(System.in);

        while (true) {
            // 显示菜单
            System.out.println("\n========== 学生管理系统 ==========");
            System.out.println("1. 添加学生");
            System.out.println("2. 查看所有学生");
            System.out.println("3. 查询学生（按学号）");
            System.out.println("4. 删除学生（按学号）");
            System.out.println("5. 统计平均分");
            System.out.println("6. 退出");
            System.out.println("==================================");
            System.out.print("请选择：");

            int choice = scanner.nextInt();
            try {
                if (choice == 1) {
                    // 添加学生
                    System.out.print("请输入学号：");
                    int id = scanner.nextInt();
                    System.out.print("请输入姓名：");
                    String name = scanner.next();
                    System.out.print("请输入成绩：");
                    double score = scanner.nextDouble();
                    Student student = new Student(id, name, score);
                    students.add(student);
                    System.out.println("添加成功！");

                } else if (choice == 2) {
                    // 查看所有学生
                    if (students.isEmpty()) {
                        System.out.println("暂无学生信息");
                    } else {
                        System.out.println("\n========== 学生列表 ==========");
                        for (int i = 0; i < students.size(); i++) {
                            students.get(i).showInfo();
                        }
                    }

                } else if (choice == 3) {
                    // 按学号查询
                    System.out.print("请输入查询学生的学号：");
                    int searchID = scanner.nextInt();
                    boolean found = false;
                    for (int i = 0; i < students.size(); i++) {
                        if (students.get(i).getId() == searchID) {
                            students.get(i).showInfo();
                            found = true;
                            break;
                        }
                    }
                    if (!found) {
                        System.out.println("未找到学号为：" + searchID + "的学生。");
                    }

                } else if (choice == 4) {
                    // 按学号删除
                    System.out.print("请输入要删除的学生学号：");
                    int deleteId = scanner.nextInt();
                    boolean found = false;
                    for (int i = 0; i < students.size(); i++) {
                        if (students.get(i).getId() == deleteId) {
                            students.remove(i);
                            System.out.println("删除成功");
                            found = true;
                            break;
                        }
                    }
                    if (!found) {
                        System.out.println("未找到学号为：" + deleteId + "的学生。");
                    }

                } else if (choice == 5) {
                    // 统计平均分
                    if (students.isEmpty()) {
                        System.out.println("暂无学生信息，无法统计");
                    } else {
                        double sum = 0;
                        for (int i = 0; i < students.size(); i++) {
                            sum = sum + students.get(i).getScore();
                        }
                        double average = sum / students.size();
                        System.out.println("平均分为：" + average);
                    }

                } else if (choice == 6) {
                    System.out.println("谢谢使用");
                    break;
                }
            } catch (Exception e) {
                System.out.println("输入错误，请输入1-6的数字");
            }
        }
        scanner.close();
    }
}
### 主程序代码解析

| 代码段 | 作用 |
|--------|------|
| `ArrayList<Student> students = new ArrayList<>()` | 存储学生对象的集合 |
| `while(true)` | 无限循环显示菜单 |
| `scanner.nextInt()` | 获取用户输入的选择 |
| `try-catch` | 捕获输入异常，防止程序崩溃 |
| `students.isEmpty()` | 判断集合是否为空 |
| `students.get(i).showInfo()` | 调用学生对象的显示方法 |
| `students.remove(i)` | 删除指定索引的学生 |
| `scanner.close()` | 关闭Scanner，释放资源 |
